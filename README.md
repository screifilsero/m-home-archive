# m-home-archive

## 项目简介

**m-home-archive** 是一个用于归档与发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名、网站或品牌，仅作为通用 HTML 资源集合的存储与展示空间。所有页面均为独立、自包含的静态文件。

## 目录说明

```
m-home-archive/
├── index.html          # 仓库总览页面（可选）
├── pages/              # 存放所有归档 HTML 页面
│   ├── example-1.html
│   ├── example-2.html
│   └── ...
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── img/
└── README.md           # 本文件
```

- `pages/`：所有归档页面的主目录。
- `assets/`：存放页面引用的公共静态资源，便于统一维护。

## 页面归档说明

- 每个 HTML 页面应保持独立完整性，可脱离仓库单独打开。
- 页面引用资源建议使用相对路径，或存放在 `assets/` 目录下。
- 页面命名建议使用小写英文，以连字符分隔，如 `my-page.html`。
- 如需添加新页面，请将文件放入 `pages/` 目录，并在必要时更新 `index.html` 中的导航链接。
